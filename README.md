# isitchristmas written by golang

## Original 
- https://isitchristmas.com/

## How to use
```bash
git clone git@github.com:yoshiyuki-140/isitchristmas-go.git
cd isitchristmas-go
docker compose up -d
```

Then, access to `localhost:8080`.
If it is christmas for you, that page show an message "Yes"
